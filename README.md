添加机器人到群组 | 公告频道 | 问题反馈

🌟 核心功能
身份过滤：自动拦截「个人主页/简介」中带有链接、频道号的恶意用户。

消息清理：实时拦截并删除「发送频道引用」或「跳转链接」的广告消息。

零管理负担：无需复杂的设置命令，拉入群组并提升为管理员即可生效。

🚀 使用指南
点击 @no_linking_bot 选择 “添加到群组”。

授予机器人 “删除消息” 和 “封禁用户” 的管理员权限。

机器人将自动开始守护群组清净。
