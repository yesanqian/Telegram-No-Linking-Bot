### 🌟 核心功能

* **身份过滤**：自动拦截「个人主页/简介」中带有链接、频道号的恶意用户。
* **消息清理**：实时拦截并删除「发送频道引用」或「跳转链接」的广告消息。
* **零管理负担**：无需复杂的设置命令，拉入群组并提升为管理员即可生效。

### 🚀 使用指南

1.  点击 [@no_linking_bot](https://t.me/no_linking_bot) 选择 **“添加到群组”**。
2.  授予机器人 **“删除消息”** 和 **“封禁用户”** 的管理员权限。
3.  机器人将自动开始守护群组清净。

### 📊 为什么选择我们？

* **极致轻量**：专注于链接与广告拦截，不干扰正常聊天。
* **即时响应**：毫秒级扫描，在广告发出的一瞬间完成清理。
* **持续更新**：针对不断进化的垃圾广告手段持续优化过滤规则。
